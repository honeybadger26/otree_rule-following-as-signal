You will now use the die and the cup. You have to roll the die using the cup, peek under the cup, and report the die-roll outcome.

 

If you report a 1, you will earn 0 points.

If you report a 2, you will earn 50 points.

If you report a 3, you will earn 100 points.

If you report a 4, you will earn 150 points.

If you report a 5, you will earn 200 points.

If you report a 6, you will earn 250 points.
